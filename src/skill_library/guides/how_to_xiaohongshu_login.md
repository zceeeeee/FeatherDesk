# 如何实现小红书验证码登录

## 流程

1. 打开 `https://www.xiaohongshu.com/login`
2. 如果出现登录按钮，先打开登录面板
3. 自动填写用户提供的手机号
4. 自动勾选用户协议、隐私政策和儿童/青少年个人信息保护规则
5. 自动点击获取验证码
6. 用户在浏览器中手动输入短信验证码

## 使用示例

```python
from skill_library.others.xiaohongshu_login import run

run("13800138000")
```

自然语言示例：

```text
登录小红书，手机号 13800138000
```

## 注意

该技能只完成验证码发送前的自动化准备，不会自动输入短信验证码，也不会在最终登录完成前保存 cookie。
